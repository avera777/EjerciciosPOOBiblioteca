#  Ejercicio POO - Biblioteca en C#

Este proyecto implementa un sistema básico de gestión de biblioteca utilizando Programación Orientada a Objetos (POO) en C#.

##  Funcionalidades

- Registro de materiales (Libros y Revistas)
- Visualización de materiales disponibles
- Préstamo de materiales a usuarios
- Manejo de errores al intentar prestar materiales ya prestados
- Devolución de materiales

##  Conceptos aplicados

- Abstracción (Clase `Material`)
- Herencia (`Libro` y `Revista`)
- Interfaces (`IPrestable`)
- Encapsulamiento
- Polimorfismo
- Manejo de excepciones
